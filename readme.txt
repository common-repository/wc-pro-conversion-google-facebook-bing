WC Pro Conversion allows you to add Facebook Pixel, Google and Bing Conversion code into your website for whole shop, category, or the product level.

== Description ==

WC Pro Conversion is the most flexible and complete solution for your conversion code for all platforms including Facebook Pixel Code, Google Conversion and Bing Conversion code. 

This plugin gives you the flexibility to add conversion codes to your shop at different levels, like you can assign a single conversion code for each platform in the whole shop, or you can assign conversion code to each category in your shop, or even upto the product level. If you enter conversion code to a specific product, it will have highest priority, medium level priority is assigned to categories and the least priority is assigned to shop conversion code.

It also gives you option to either put your code in footer or header.


== Installation ==

1. Extract the zip folder and just drop the folder "wc-category-conversion-code" in wordpress plugins directory wp-content/plugins/
2. Go to plugins interface and activate the plugin.
3. Go to "Woocommerce" menu and click on submenu "Conversion Code" for settings
4. Set default code for facebook,bing and yahoo.
